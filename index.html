How to Make a Simple Cup of Tea

  1. Boil water.
  2. Prepare the tea:
    1. Choose your favorite tea bag or loose-leaf tea.
    2. Place it in a cup or teapot.
  3. Pour the hot water over the tea.
  4. Let it steep for a few minutes.
  5. Remove the tea bag or strain the tea.
  6. Enjoy your tea with optional add-ins like honey or lemon.

How to use TypeScript to get a value of an interface property


interface MyInterface {
    id: number;
    name: string;
    properties: string[];
}

const myObject: MyInterface = {
    id: 1,
    name: 'foo',
    properties: ['a', 'b', 'c']
};

function getValue(value: keyof MyInterface) {
    return myObject[value];
}

console.log(getValue('id')); // 1
console.log(getValue('name')); // 'foo'
